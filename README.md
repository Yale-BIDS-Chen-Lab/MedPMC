# MedPMC

MedPMC is an ongoing effort to build large-scale medical vision-language resources from biomedical literature. This repository contains the codebase for dataset curation, preprocessing, model training, and evaluation.

## Resources

### Dataset
- **The first version of the MedPMC Dataset (June 2024 PMC Baseline):** ~11 million medical image-text pairs extracted from PubMed Central (PMC) articles. 
  - [Hugging Face](https://huggingface.co/datasets/Yale-BIDS-Chen/medpmc-11m-dataset_jun24_baseline)

### Models
- **MedPMC-CLIP:** Medical vision-language model trained on MedPMC-11M.
  - [Hugging Face](https://huggingface.co/Yale-BIDS-Chen/medpmc-clip-l-14_jun24_v1)

## Status

Additional datasets, models, benchmarks, training recipes, and documentation will be released soon.
